Numerical Project

The numerical project is a larger assignment that is handed in in the second half of the semester, applying many of the physical and numerical methods taught earlier in the course. The scope of the project involves both wavephysics and fluidmechanics as well as advanced numerical methods. The project is to be handed in March 20th. In addition both the project and the assignments are to be handed in using Jupyter Notebook(a graphical interface packaged with Anaconda).
